よくある質問

OWD講習についてのご質問にお答えします

完全初心者でも大丈夫ですか？

+

はい、全く問題ありません。OWD講習は初心者の方を対象としたコースです。eラーニングで基礎知識をしっかり学んでから実習に進むので、安心して受講いただけます。これまでの受講生の約8割が完全初心者の方でした。

eラーニングはどのくらい時間がかかりますか？

+

個人差はありますが、一般的に2〜4時間程度です。24時間いつでもアクセス可能で、途中で中断・再開できるので、自分のペースで学習を進められます。理解できるまで何度でも繰り返し学習できます。

ライセンスはどのくらいで発行されますか？

+

海洋実習終了後、約1〜2週間でPADI本部からライセンスカードが発行されます。それまでの間は仮認定証をお渡しするので、すぐにダイビングを楽しんでいただけます。

3日間連続で受講する必要がありますか？

+

いいえ、連続でなくても大丈夫です。eラーニング→プール実習→海洋実習の順番は守っていただきますが、それぞれ別の日に受講可能です。お仕事やご都合に合わせてスケジュール調整いたします。

海洋実習は雨天でも実施しますか？

+

雨天の場合でも、海況が安全であれば実施いたします。ただし、波が高い、風が強いなど安全に実習できない場合は延期となります。その場合、キャンセル料はかかりません。

冬でも海洋実習は受けられますか？

+

はい、年間を通じて実施しています。冬場はドライスーツを使用するため、海水温に関係なく快適にダイビングできます。冬の海は透明度が高く、とても綺麗です。

器材は購入する必要がありますか？

+

講習中は全てレンタル器材をご利用いただけるので、購入の必要はありません。様々なメーカーの器材を試せるので、後で購入される際の参考にもなります。器材購入をご希望の場合はご相談ください。

持参するものはありますか？

+

水着、タオル、着替え、サンダル（滑りにくいもの）をご持参ください。プール実習では温水プール用の水着、海洋実習では日焼け止めもお忘れなく。詳しいリストは申し込み後にお送りします。

レンタル器材は清潔ですか？

+

はい、すべての器材は使用後に専用洗剤で清洗・消毒し、定期的にメンテナンスを行っています。特にレギュレーター（呼吸器）は厳格な点検・整備を実施しており、安心してご利用いただけます。

追加料金はかかりませんか？

+

基本的な講習費用（¥53,900）に含まれない費用は、レンタル器材代（¥8,000/日）、昼食代、交通費のみです。隠れた追加料金は一切ございません。明確な料金体系で安心してお申し込みいただけます。

分割払いは可能ですか？

+

クレジットカードでお支払いの場合、カード会社の分割払い・リボ払いをご利用いただけます。また、2回払い（申し込み時と受講時）も承っておりますので、お気軽にご相談ください。

キャンセル料はいつからかかりますか？

+

7日前までは無料、6〜3日前30%、2日前〜前日50%、当日100%となります。ただし、海況不良による開催中止や体調不良による当日キャンセルは別途ご相談させていただきます。

泳ぎが苦手でも大丈夫ですか？

+

基本的な泳力（25m程度）は必要ですが、競泳のように速く泳ぐ必要はありません。ダイビングは泳ぐというより、浮力を使って水中を移動します。不安な方は事前にプールで練習することをお勧めします。

年齢制限はありますか？

+

10歳以上であればご受講いただけます。上限年齢は設けておりませんが、健康状態に問題がないことが条件です。60歳以上の方は医師の診断書が必要になる場合があります。

持病がある場合は受講できませんか？

+

病気の種類によりますが、多くの場合は医師の診断書があれば受講可能です。心臓病、喘息、糖尿病、高血圧などをお持ちの方は、事前にご相談ください。安全第一で判断いたします。

生理中でも海洋実習は受けられますか？

+

はい、問題ありません。タンポンを使用していただければ、通常通り実習を受けられます。体調が優れない場合は日程変更も可能ですので、無理をせずにご相談ください。

一人での参加は心配です...

+

ご安心ください。お一人での参加の方も多くいらっしゃいます。少人数制の講習なので、インストラクターがしっかりサポートしますし、同期の受講生とも仲良くなれます。ダイビング仲間ができるチャンスでもあります。

講習終了後のサポートはありますか？

+

はい、ライセンス取得後も定期的にファンダイビングを開催しています。また、AOWなどのスキルアップコースもご案内しております。長くダイビングを楽しんでいただけるよう、継続的にサポートいたします。

何名くらいで講習を行いますか？

+

安全で質の高い講習を提供するため、インストラクター1名に対して最大4名までの少人数制で実施しています。お一人お一人に丁寧に指導できる人数設定です。
